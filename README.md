# 🚀 EventChain - Smart Event Management Dashboard

A modern and professional **Event Management Dashboard** built using **HTML, CSS, and JavaScript**.
EventChain provides an interactive admin-style interface to manage events, clubs, blockchain records, analytics, and user settings — all within a clean and responsive dashboard design.

---

## 📌 Project Overview

EventChain is a web-based dashboard designed to simplify event and club management inside organizations or colleges.

The platform allows users to:

* View dashboard overview and stats
* Manage events and clubs
* Track blockchain-based records
* Analyze engagement metrics
* Configure profile and system settings

This project focuses on **UI/UX design**, **dashboard layouts**, and **dynamic frontend functionality** using vanilla JavaScript.

---

## ✨ Features

### 🧭 Dashboard Overview

* Statistics cards (Events, Clubs, Participants, Blockchain Records)
* Upcoming events section
* Recent activities feed

### 📅 Event Management

* View all events
* Event filtering options
* Create new events using modal form
* Dynamic event display

### 👥 Club Management

* Manage clubs and memberships
* Club cards generated dynamically

### 🔗 Blockchain Records

* Blockchain visualization section
* Hash algorithm info display
* Immutable event tracking concept

### 📊 Analytics

* Event attendance insights
* Club participation overview
* Chart placeholders for future integration

### ⚙️ Settings

* Profile settings form
* Toggle options for blockchain auto-recording

### 🔔 Extra UI Components

* Sidebar navigation
* Top search bar
* Notifications & messages icons
* Toast notifications
* Modal popup for creating events
